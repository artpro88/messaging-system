# How to Create PostgreSQL Database on Fly.io\n\n## Where to Click\n\n1. **Go to**: https://fly.io/app/dashboard\n2. **Look for**: Button in top right that says **\"New App\"** or **\"+\"**\n3. **Click it**\n\n## What You'll See\n\nA menu with options like:\n- Database\n- App\n- Organization\n- etc.\n\n## Steps\n\n1. Click **\"Database\"** from the menu\n2. You'll see database options:\n   - PostgreSQL\n   - MySQL\n   - Redis\n   - etc.\n\n3. Click **\"PostgreSQL\"**\n\n4. **Configuration Dialog** will appear:\n   - **App Name**: Type `messaging-db`\n   - **Region**: Select `London (lhr)`\n   - **Configuration**: Click the **Development** option (free tier)\n     - Shows: \"Development - Single node, 1x shared CPU, 256MB RAM, 1GB disk\"\n\n5. Click **\"Create Database\"** button\n\n6. **Wait** for 2-3 minutes while it provisions\n\n## What You'll Get\n\nAfter it finishes, you'll see a page showing:\n\n```\nConnection string:\npostgresql://postgres:PASSWORD@messaging-db.flycast:5432/postgres\n\nOr separately:\n- Host: messaging-db.flycast\n- Port: 5432\n- Database: postgres\n- User: postgres\n- Password: [long random string]\n```\n\n## IMPORTANT: Copy This!\n\n**Copy the full connection string** - you need it for the next step.\n\nIt looks like:\n```\npostgresql://postgres:ACTUAL_PASSWORD_HERE@messaging-db.flycast:5432/postgres\n```\n\n## Next Step\n\nGo to `FLY_DEPLOYMENT_STEPS.md` and follow **Step 2: Set Database Secret**\n\nYou'll paste this connection string as the `DATABASE_URL` secret.\n\n## Can't Find the \"New App\" Button?\n\n**Alternative way**:\n\n1. Go to https://fly.io/app/dashboard\n2. Look at left sidebar - you'll see a list of your apps\n3. At the **top** of that list, there's a **\"+\"** or **\"Create\"** button\n4. Click it\n5. Follow the steps above\n\n## If It Still Asks for Configuration\n\nWhen you select Development configuration, if it asks for settings:\n\n- **VM Size**: shared-cpu-1x\n- **RAM**: 256MB\n- **Storage**: 1GB\n- **High Availability**: NO (uncheck if offered)\n\nThen click **Create**.\n\n## Screenshots Would Help\n\nIf you're still having trouble finding it, let me know and I can guide you through finding the exact button location.\n"}
